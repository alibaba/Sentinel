# Sentinel Dashboard Frontend

## Env Requirement

- Node.js > 6.x
- Node.js < 12.x

## Code Guide

- [Code Style Guide for HTML/CSS](https://codeguide.bootcss.com/)
- [Airbnb JavaScript Style Guide](https://github.com/airbnb/javascript/tree/es5-deprecated/es5)

## Install Packages

```
npm install
```

## Start Development

```
npm start
```

## Build for production

```
npm run build
```
